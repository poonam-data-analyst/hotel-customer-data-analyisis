# 🏨 Hotel Customer Intelligence & Revenue Optimization

> 🚀 End-to-End python Project | Customer Segmentation | Booking Prediction | Revenue Forecasting

## 📌 Overview
This project delivers a **comprehensive, end-to-end data science solution** on a real-world hotel dataset (2015–2018), transforming raw customer data into **actionable business insights**.

It integrates **exploratory data analysis, machine learning, and business strategy** to solve key hospitality challenges:
- Who are the most valuable customers?
- Which bookings are likely to cancel?
- How can revenue be maximized?

## 🎯 Business Objectives
✔ Identify high-value customer segments  
✔ Reduce booking cancellations using predictive modeling  
✔ Forecast and optimize revenue drivers  
✔ Enable data-driven decision-making in hospitality  

## 📊 Dataset Snapshot
- 📍 Location: Lisbon, Portugal  
- 👥 Records: 83,590 customers  
- 📌 Features: 31 variables  
- 🔄 Data Type: Mixed (Categorical, Numerical, Boolean)
